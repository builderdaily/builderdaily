# Builder Daily

Daily AI opportunity analysis for builders. The site turns AI and developer
tooling signals into business-webapp opportunities, with source links, scoring,
and deep dives for the top opportunity.

## Local Preview

This is a static site. Serve the repository root with any static file server:

```powershell
python -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173/
```

## Custom Domain

The intended production domain is:

```text
builderdaily.news
```
