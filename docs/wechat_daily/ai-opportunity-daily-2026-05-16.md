## 技巧/方法论提醒

### 1）哥飞做站方法论/可转化机会

- **价格是转化触发器**：2026-05-16 11:33-11:34｜赫兹的朋友们~出海交流｜Yifei易飞/droidHZ｜“看见哥飞分享，大半夜看见价格就买了”。说明不是所有机会都靠“新词”，清晰价格和可复制路径能直接促成购买。
- **免费用户和付费用户要分 API 层级**：2026-05-16 20:36｜哥飞的朋友们⑩｜布衣｜“免费试用用 kie，付费用官方”。这条可直接用于 AI 视频站、图片站、语音站：免费额度用低成本/可失败模型，付费任务走官方/稳定线路。
- **可转化方向**：围绕“API 选择、失败率、成本覆盖、免费额度边界、付费用户保障”做页面和工具，比单纯追新模型更接近变现。

### 2）赫兹 Prompt 素材获取 / GitHub 导流：放技巧，不当新站机会

昨日记录里再次确认：从 GitHub、竞品、推特大 V 获取图片/视频 prompt，或做 GitHub 集合项目导流，本质是**素材获取和分发技巧**。它可以服务已有图片站/视频站/Prompt 库，但不应单独包装成“新词新站机会”，除非另有搜索量/付费需求证据。

可借鉴做法：
- 给已有图片/视频站补“Prompt examples / templates / style library”。
- GitHub 开源一个 curated collection，给主站导流和做外链资产。
- 不要把“Prompt 模板站”当核心商业模式，除非绑定具体模型、具体任务、具体付费场景。

### 3）先高质量报告，后 SaaS

电商测款方向给出的路径值得复用：不先做复杂数据库/自动化，而是先用固定评分维度做“像顾问交付物”的报告，收钱验证。适用于 GEO 审计、AI 视频 API 选型、工具站安全审计等方向。

## 被降权/剔除方向

1. **ChatGPT 个人理财 / 接银行账户**  
   公众号新闻多，但这是 OpenAI + 金融机构级别的强监管平台机会；独立开发者不适合直接做“接银行账户”的产品。最多做解读页，不列 Top。

2. **上海电信 Token 话费套餐**  
   公众号/账号会话 AGENT橘提到“1 元 25 万 token、话费账单扣”。这是运营商渠道/地区政策，不是独立开发者可控新站机会。可观察 API 价格锚点。

3. **Articraft 可动 3D 资产代码智能体**  
   GitHubStore 链接有技术新鲜度，但昨天缺少群聊需求、价格/失败/教程痛点；3D 资产生成还偏重算力和专业链路，先降权。

4. **Claude Code 日记/复盘转 HTML 报告**  
   有公众号案例，但群里也出现“市面上很多类似、开源多、难变现”的判断。可作为个人自动化模板，不建议单独做新站。

## 今日建议执行顺序

1. 先做 **AI 视频 API 稳定性/alternatives**：需求最硬，有失败、有付费分层、有哥飞方法论支撑。  
2. 同步做 **Codex Doctor**：页面多、长尾多、当天证据密集，适合快速铺 SEO。  
3. 用 **收款安全 checklist** 做 B2B 服务入口：最容易拿到咨询/审计钱。  
4. 电商测款报告和 GEO 审计都先人工交付，不急着自动化。
